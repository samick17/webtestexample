## Example of simple-web-automator

####

    npm install simple-web-automator --save

#### Run

`
const simpleWebAutomator = require('simple-web-automator');
simpleWebAutomator
.setwd('./testscripts')
.run('testcase1');
`

#### Run step-by-step

`
const simpleWebAutomator = require('simple-web-automator');
simpleWebAutomator
.setwd('./testscripts')
.run('testcase1');
`
